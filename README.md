# node-ebot
## A eBot implementation in pure node.js
---
### Planned features
- Server Management
- Configuration (Web/Console)
- Match Making
    - formats (BO1, BO3, etc.)
    - map voting (veto, pick, random) [plugin]
    - scorebot [plugin]
    - statistics [plugin]
    - team flags
    - pause/unpause
    - overtime (MR6/MR10)
    - Full Configurable
- Tournement/Lige Managment
- Plugin API
